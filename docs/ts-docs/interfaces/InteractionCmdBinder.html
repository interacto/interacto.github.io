Interface InteractionCmdBinder<C, I, D>

The binder API that already knows the type of UI command and the user interaction to use.

Type parameters

  • C: Command

    The type of the produced UI commands

  • I: Interaction<D>

    The type of the user interaction

  • D: InteractionData

    The type of the interaction data of the user interaction
