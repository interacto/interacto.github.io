textInputBinder

  • textInputBinder(timeout?: undefined | number): InteractionUpdateBinder<Interaction<WidgetData<HTMLInputElement | HTMLTextAreaElement>>, WidgetData<HTMLInputElement | HTMLTextAreaElement>>
    • Defined in src/api/binding/Bindings.ts:117

    Creates a binding that uses a text interaction. This binder takes as argument a timeout value: using this text writing interaction, a user can write a sequence of letters and then stops for more than x seconds (x is the value of timeout). After this delay the binding executes the command. This is a mainstream optimisation that many text processing tools implement to limit the number of editing actions.

    Parameters

    • Optional timeout: undefined | number

      The timeout after which the interaction stops and the command produced.

    Returns InteractionUpdateBinder<Interaction<WidgetData<HTMLInputElement | HTMLTextAreaElement>>, WidgetData<HTMLInputElement | HTMLTextAreaElement>>
