Interacto

Interacto is a front-end framework for processing user interface events. With Interacto developers handle user interactions (DnD, drag-lock, double-click, etc.) instead of low-level UI events. Interacto allows developers to define how to turn a given user interaction into a (undoable) UI command.


Howto

JavaFX and Maven

<dependency>
    <groupId>io.github.interacto</groupId>
    <artifactId>interacto-java-api</artifactId>
    <version>${interacto.version}</version>
</dependency>
<dependency>
    <groupId>io.github.interacto</groupId>
    <artifactId>interacto-javafx</artifactId>
    <version>${interacto.version}</version>
</dependency>
